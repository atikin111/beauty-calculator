# Beauty Calculator

Статический одностраничный бьюти-калькулятор.

## Локальный запуск

```bash
python main.py
```

Открыть:

```text
http://127.0.0.1:8000/index.html
```

## Публикация

Проект можно размещать как статический сайт на `GitHub Pages`, `Netlify` или `Vercel`.
Главная страница находится в файле `index.html`.
